#### 1. 一元二次方程（初中代数之魂）

**初中定义**  
ax² + bx + c = 0 (a ≠ 0)

```python
x = symbols('x')

# 经典题型全家福
equations = [
    "x² - 5x + 6 = 0",          # 配方 + 因式分解
    "2x² + 4x - 30 = 0",        # 系数化简
    "x² - 6x + 7 = 0",          # Δ < 0 无实根
    "3x² - 12x + 12 = 0",       # 根相等
    "(x-1)² = 8",               # 完全平方
]

print("一元二次方程求根大乱斗".center(60, "—"))
for eq in equations:
    solution = solve(eq, x)
    disc = sp.discriminant(eq)
    print(f"{eq:20}  →  根 = {solution}    Δ = {disc}")
```

输出示例：
```
x² - 5x + 6 = 0       →  根 = [2, 3]        Δ = 25
2x² + 4x - 30 = 0      →  根 = [-5, 3]        Δ = 256
x² - 6x + 7 = 0        →  根 = [3 ± √2]      Δ = 8
```

#### 3. 分式方程（初中最容易丢分的题型）

**增根警告**：使分母为 0 的解必须舍去！

```python
# 经典分式方程
eqs = [
    "2/x + 3/(x+1) = 1",           # 标准型
    "(x+3)/(x-2) = (x+1)/(x-4)",    # 交叉相乘
    "1/(x-1) + 1/(x-2) = 3/4",     # 通分大法
]

print("分式方程秒杀".center(50, "—"))
for eq in eqs:
    sol = solve(eq, x)
    valid = [s for s in sol if sp.denom(eq.lhs.subs(x,s)) != 0]
    print(f"{eq:30}  →  解: {valid}")
```

#### 4. 一元高次方程（初中选修/奥数压轴）

```python
# 能用换元、因式分解、降次的全家福
high_orders = [
    "x⁴ - 5x² + 4 = 0",            # 换元 t = x²
    "(x² - 3x + 1)² - 8(x² - 3x + 1) + 12 = 0",   # 换元降次
    "x³ - 6x² + 11x - 6 = 0",       # 有理根定理
]

print("一元高次方程降次大法".center(50, "—"))
for eq in high_orders:
    roots = solve(eq, x)
    print(f"{eq:35}  →  根: {roots}")
```

#### 5. 二元一次方程组（初中方程组之王）

```python
x, y = symbols('x y')

# 三种方法现场表演
system = [2*x + 3*y - 17, x + 5*y - 24]

print("二元一次方程组三种死法".center(50, "="))
print("① 代入消元法")
sol1 = solve(system, (x, y))

print("② 加减消元法（手算最快）")
sol2 = solve([4*x + 6*y - 34, 3*x + 15*y - 72], (x, y))

print("③ NumPy 矩阵法（研究生标配）")
A = np.array([[2, 3], [1, 5]])
b = np.array([17, 24])
solution = np.linalg.solve(A, b)
print(f"NumPy 解: x = {solution[0]:.0f}, y = {solution[1]:.0f}")

print(f"最终答案: x = 3, y = 5")
```
