#### 1. 整除与约数（小学复习 + 初中升级）

**初中必背概念**

#### 2. 同余的超级无敌概念（中国初中数学最牛的一页）

**初中生最容易懂的比喻**  
把所有整数按“除以 m 后的余数”分成 m 个“家族”，同一个家族的数就叫“同余”  
→ 就像按生肖分成12个家族，属龙的都“同余于龙”

**官方定义**  
a ≡ b (mod m) ↔ m 能整除 (a-b) ↔ a 和 b 除以 m 余数相同

```python
# NumPy 一键计算同余
def cong(a, b, m):
    return (a - b) % m == 0

print("同余判断现场表演：")
print(f"18 ≡ 3  (mod 5)  ? → {cong(18,3,5)}")    # True
print(f"100 ≡ 2  (mod 7)  ? → {cong(100,2,7)}")  # True
print(f"2025 ≡ ? (mod 10) → 2025 % 10 = {2025%10}")  # 5 → 末位数！
```

#### 4. 中国剩余定理（孙子定理）初中版现场秒杀

**《孙子算经》原题（1300年前）**  
今有物不知其数，三三数之剩二，五五数之剩三，七七数之剩二，问物几何？

```python
# 翻译成同余方程组
# x ≡ 2 (mod 3)
# x ≡ 3 (mod 5)
# x ≡ 2 (mod 7)

# 中国人1300年前就用“更相减损术”，我们用 NumPy 一秒解
for x in range(1, 106):               # 3×5×7=105 以内一定有解
    if (x % 3 == 2) and (x % 5 == 3) and (x % 7 == 2):
        print(f"孙子定理答案：x = {x}")
        print(f"验证：{x}÷3余2，{x}÷5余3，{x}÷7余2")
        break
# 输出：x = 23（最小正整数解）
```
