#### 1. 平面直角坐标系（概念 + 为什么这么重要）

**初中生最容易懂的概念**  
想象一张无限大的格子纸，横线叫 x 轴，竖线叫 y 轴，它们在原点 O 互相垂直。  
任何一个点的位置，都可以用“向右走几步，向上走几步”两个数字完全确定，这就是它的坐标 (x, y)。

**象限口诀（背了就再也不忘）**  
- 右上Ⅰ：正正得正  
- 左上Ⅱ：负正得负  
- 左下Ⅲ：负负得正  
- 右下Ⅳ：正负得负

```python
fig, ax = plt.subplots(figsize=(12,12))
ax.set_xlim(-7,7); ax.set_ylim(-7,7)
ax.axhline(0,color='k',lw=2); ax.axvline(0,color='k',lw=2)

```

#### 2. 两点之间的距离公式（为什么是这个公式？）

**概念讲解**  
两点之间的最短路径就是连线。  
用勾股定理：  
横着走了 |x₂-x₁|，竖着走了 |y₂-y₁|，  
斜边就是 √[(x₂-x₁)² + (y₂-y₁)²]

```python
def distance(p1, p2):
    return np.sqrt((p2[0]-p1[0])**2 + (p2[1]-p1[1])**2)

A, B = np.array([1, 2]), np.array([5, 5])
d = distance(A, B)
print(f"点 A{ A } 到点 B{ B } 的距离 = {d:.2f}")   # 5.00

# 现场画出来看勾股定理
plt.figure(figsize=(8,8))
plt.plot([A[0],B[0]],[A[1],B[1]],'b-',linewidth=3,label=f'距离={d:.2f}')
plt.plot([A[0],B[0]],[A[1],A[1]],'r--')   # 横边
plt.plot([B[0],B[0]],[A[1],B[1]],'r--')   # 竖边
plt.plot(A[0],A[1],'ro',markersize=12,label='A(1,2)')
plt.plot(B[0],B[1],'go',markersize=12,label='B(5,5)')
plt.grid(True); plt.axis('equal')
plt.legend(); plt.title('两点间距离公式的勾股定理来源')
plt.show()
```

#### 3. 点到直线的距离（初中最难但最重要公式）

**概念讲解（初中生也能听懂版）**  
1. 先把直线写成 Ax + By + C = 0 的形式  
2. 把点 (x₀,y₀) 代进去，看它“超了多少” → |Ax₀ + By₀ + C|  
3. 再除以斜边的长度 √(A²+B²) → 就是垂直距离！

```python
# 公式完整版
def dist_point_to_line(A, B, C, P):
    x0, y0 = P
    return np.abs(A*x0 + B*y0 + C) / np.sqrt(A**2 + B**2)

# 例题：点 P(3,1) 到直线 3x - 4y + 2 = 0 的距离
d = dist_point_to_line(3, -4, 2, (3,1))
print(f"距离 = {d:.3f}")   # 0.200

# 可视化（看垂直关系）
x = np.linspace(-1,5,400)
y = (3*x + 2)/4
P = np.array([3,1])

plt.figure(figsize=(10,8))
plt.plot(x,y,'blue',lw=3,label='3x-4y+2=0')
plt.plot(3,1,'ro',markersize=12,label='P(3,1)')

# 垂足坐标（初中不要求会算，但我们偷偷告诉你）
foot_x = (B*(B*3 - A*1) - A*C) / (A**2 + B**2)
foot_y = (A*(-B*3 + A*1) - B*C) / (A**2 + B**2)
plt.plot(foot_x, foot_y, 'go', markersize=10)
plt.plot([3,foot_x],[1,foot_y],'r--',lw=2)

plt.text(3.1,1.2,f'距离={d:.3f}',color='red',fontsize=14,weight='bold')
plt.axis('equal'); plt.grid(True,alpha=0.3)
plt.legend(); plt.title('点到直线距离的几何意义（垂线段最短）')
plt.show()
```

#### 4. 圆的方程（标准方程 + 一般方程）

**概念讲解**  
圆就是到某个固定点（圆心）距离相等的点的集合！

- 标准方程：(x−h)² + (y−k)² = r²  
- 一般方程：x² + y² + Dx + Ey + F = 0（配方后可得到圆心半径）

```python
# 标准方程画圆
fig, ax = plt.subplots(figsize=(10,10))
ax.set_xlim(-8,8); ax.set_ylim(-8,8)
ax.grid(True,alpha=0.3); ax.set_aspect('equal')

circles = [
    ((0,0), 5, 'red',    '圆心在原点'),
    ((3,2), 4, 'blue',   '圆心(3,2)'),
    ((-2,-1),3,'green',  '圆心(-2,-1)')
]

for (h,k), r, col, lab in circles:
    circle = Circle((h,k), r, fill=False, color=col, lw=3, label=lab+f' 方程:(x{h:+.0f})²+(y{k:+.0f})²={r}²')
    ax.add_patch(circle)
    ax.plot(h,k,'o',color=col,markersize=8)
    ax.text(h+0.5,k,f'({h},{k})',fontsize=12)

ax.set_title('圆的标准方程完整概念图',fontsize=20)
ax.legend(loc='upper left')
plt.show()
```
