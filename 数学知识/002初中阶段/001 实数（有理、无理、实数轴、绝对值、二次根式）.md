#### 1. 有理数 vs 无理数（初中第一次见√2就懵了？现在让你彻底明白）

**小学生也能懂的定义**  
- 有理数：能写成分数形式的数（包括整数、小数、百分数）  
- 无理数：无限不循环小数，永远写不成分数（比如 √2、π）

```python
# 有理数全家福
rational_examples = np.array([
    -5, 0, 3, 1/2, 0.75, 0.333333333333, 22/7, -8/4
])
print("有理数示例：", rational_examples)

# 无理数经典代表
irrational = np.array([
    np.sqrt(2),      # ≈1.4142135623730951
    np.sqrt(3),      
    np.pi,           # ≈3.141592653589793
    np.e,            # ≈2.718281828459045
    np.sqrt(5), 
    np.pi/4
])
print("\n无理数示例（无限不循环）:")
for x in irrational:
    print(f"{x:.15f}")
```

#### 2. 实数轴（数轴）—— 初中几何与代数的第一次完美结合

```python
# 画一条超漂亮的实数轴（初中数学书封面级别）
fig, ax = plt.subplots(figsize=(14, 6))
ax.axhline(0, color='black', linewidth=2)

# 刻度
ticks = np.array([-4, -np.sqrt(8), -2, -1, 0, 1, np.sqrt(2), 2, np.pi, 4])
ax.set_xticks(ticks)
ax.set_xticklabels([
    '-4', '-2√2', '-2', '-1', '0', '1', '√2≈1.414', '2', 'π≈3.14', '4'
], fontsize=12)

# 标注有理数（蓝色）和无理数（红色）
for t in ticks:
    color = 'blue' if t in [-4,-2,-1,0,1,2,4] else 'red'
    ax.plot(t, 0, 'o', color=color, markersize=8)
    ax.vlines(t, -0.15, 0.15, color=color, linewidth=2)

ax.text(np.sqrt(2), 0.25, '√2', fontsize=14, color='red', weight='bold')
ax.text(np.pi, 0.25, 'π', fontsize=14, color='red', weight='bold')
ax.set_title('实数轴：有理数（蓝） vs 无理数（红）\n初中数学最美的一张图', fontsize=18, pad=20)
ax.axis('off')
plt.show()
```

#### 3. 绝对值 |x| —— 初中不等式之王

```python
x = np.linspace(-5, 5, 400)
y = np.abs(x)

plt.figure(figsize=(10,6))
plt.plot(x, y, 'purple', linewidth=4, label='y = |x|')
plt.grid(True, alpha=0.3)
plt.axvline(0, color='black', linewidth=1)
plt.axhline(0, color='black', linewidth=1)
plt.title('绝对值函数图像（V形）—— 初中分段函数第一课', fontsize=16)
plt.xlabel('x'); plt.ylabel('|x|')
plt.legend(fontsize=14)
plt.xticks(range(-5,6)); plt.yticks(range(0,6))
plt.show()

# 绝对值几何意义：到原点的距离
print("验证：|3| =", np.abs(3))      # 3
print("|-4| =", np.abs(-4))          # 4
print("|√2 - 1| ≈", np.abs(np.sqrt(2)-1))
```

#### 4. 二次根式（√）—— 初中代数最难简化的一章，现在 1 秒搞定

```python
# 经典初中题型全家福
expressions = [
    "√50", "√72", "√200", "3√8", "√(1/2)", "√12 + √27", "√18 × √8"
]

print("二次根式最简形（初中必考）".center(50, "—"))
for expr in expressions:
    result = simplify(expr)
    print(f"{expr:>12}  →  {result}")
```

输出：
```
√50        →  5√2
√72        →  6√6
√200       →  10√2
3√8        →  6√2
√(1/2)     →  √2/2
√12 + √27  →  5√3
√18 × √8   →  12
```

#### 5. 二次根式的四则运算（初中压轴计算题）

```python
a = 3*np.sqrt(2)
b = 5*np.sqrt(2)
c = np.sqrt(8)     # = 2√2
d = np.sqrt(18)    # = 3√2

print("二次根式计算大乱斗：")
print(f"3√2 + 5√2 = {a + b}")                    # 8√2
print(f"5√2 - 3√2 = {b - a}")                    # 2√2
print(f"√8 × √18 = {c * d}")                     # 12
print(f"√50 ÷ √2 = {np.sqrt(50)/np.sqrt(2)}")    # 5
print(f"(3√2)² = {a**2}")                        # 18
```

#### 6. 实数大小比较终极技巧（含二次根式的无计算比较）

```python
# 经典题：比较 √2, √3, √5, 11/7, 17/10 大小（不许用计算器！）
nums = np.array([np.sqrt(2), np.sqrt(3), np.sqrt(5), 11/7, 17/10])

# 方法①：平方比较（初中最强武器）
squares = nums**2
print("直接平方比较（越大越大的顺序）：")
print("√2² =", squares[0])
print("√3² =", squares[1])  
print("(11/7)² =", (11/7)**2)
print("(17/10)² =", (17/10)**2)
print("√5² =", squares[2])
print("→ 排序结果：√2 < 11/7 < 17/10 < √3 < √5")
```
