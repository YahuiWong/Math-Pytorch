#### 1. 频率与概率（初中最重要的一次“实验→理论”飞跃）

**初中生最容易懂的概念**  
- 频率 = 做了很多次，某件事发生的次数 ÷ 总次数  
- 当实验次数 → ∞ 时，频率会稳定到一个值 → 这就是概率

```python
# 现场掷10000次硬币看频率如何趋向0.5
n_trials = 10000
coins = np.random.randint(0, 2, size=n_trials)        # 0=反面 1=正面
cumulative_positive = np.cumsum(coins)                # 到目前为止正面次数
frequencies = cumulative_positive / np.arange(1, n_trials+1)

plt.figure(figsize=(12,6))
plt.plot(frequencies, color='purple', linewidth=2)
plt.axhline(0.5, color='red', linestyle='--', linewidth=2, label='理论概率 = 0.5')
plt.title('掷硬币10000次：频率 → 概率（初中概率第一铁律）',fontsize=18)
plt.xlabel('实验次数'); plt.ylabel('正面频率')
plt.ylim(0,1); plt.grid(alpha=0.3)
plt.legend(fontsize=14)
plt.show()
```

#### 2. 古典概型（中考概率计算的全部）

**初中必背三句话**  
1. 所有结果同样可能  
2. 总结果数能数清楚  
3. 概率 = 目标结果数 ÷ 总结果数

```python
# 经典题型全家福
problems = [
    ("掷一枚骰子，朝上的点数为偶数", 3, 6),
    ("掷两枚骰子，点数之和为7", 6, 36),
    ("从红5黄3蓝2共10张牌中随机抽1张，是红牌", 5, 10),
    ("同时掷3枚硬币，全是正面", 1, 8),
]

print("古典概型秒杀表".center(50,"="))
for q, favor, total in problems:
    prob = favor / total
    print(f"{q:40} → 概率 = {favor}/{total} = {prob:.3f}")
```

#### 3. 统计图表三兄弟（条形图、折线图、扇形图）

```python
# 八年级（2）班某次数学成绩统计
scores = np.array([62,78,85,91,73,88,95,67,82,90,74,99,81,76,93,87,70,94,79,86])
labels = ['60-70','70-80','80-90','90-100']

plt.bar(labels, counts, color=['#FF9999','#66B2FF','#99FF99','#FFCC99'], edgecolor='black')

    plt.text(i, c+0.2, str(c), ha='center', fontsize=14, weight='bold')

plt.ylabel('人数')

# 折线图（频率分布）
plt.subplot(1,3,2)

plt.plot(labels, cum, 'o-', color='purple', linewidth=3, markersize=8)
plt.title('累积频率折线图')
plt.ylabel('累计人数')

# 扇形图
plt.subplot(1,3,3)
plt.pie(counts, labels=labels, autopct='%1.1f%%', startangle=90, 
        colors=['#FF9999','#66B2FF','#99FF99','#FFCC99'])
plt.title('扇形图：成绩占比')

plt.suptitle('初中统计三大图表一次看全',fontsize=22)
plt.tight_layout()
plt.show()
```

#### 4. 平均数、中位数、众数（初中统计三大“数”）

```python
data = np.array([2,3,3,4,4,4,5,6,7,10])

mean = np.mean(data)
median = np.median(data)
mode = 4   # 出现最多次

print(f"数据：{data}")
print(f"平均数（算术平均）= {mean:.2f}")
print(f"中位数（中间位置）= {median}")
print(f"众数（出现最多次）= {mode}")

# 现场演示“平均数会被极端值拉偏”
data2 = np.append(data, 100)
print(f"\n加一个100后：{data2}")
print(f"平均数暴涨 → {np.mean(data2):.2f}（平均数不稳健）")
print(f"中位数几乎不变 → {np.median(data2):.2f}（中位数更稳健）")
```
