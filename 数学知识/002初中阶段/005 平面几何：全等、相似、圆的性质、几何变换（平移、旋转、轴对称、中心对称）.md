#### 1. 全等三角形（概念 + 五大判定法 + 动图演示）

**初中生最容易懂的概念**  
两个三角形能完全重合（大小、形状、方向都一样）→ 全等  
就像两个完全一样的积木

**五大判定口诀（背会就无敌）**  
- SSS：三边对应相等  
- SAS：两边夹一角  
- ASA：两角夹一边  
- AAS：两角和第三边  
- HL：直角三角形的斜边+一条直角边

```python
# 现场演示 SSS 全等
A1, B1, C1 = np.array([0,0]), np.array([6,0]), np.array([3,4])
A2, B2, C2 = np.array([8,1]), np.array([2,1]), np.array([5,5])

fig, ax = plt.subplots(figsize=(12,6))
ax.set_xlim(-1,12); ax.set_ylim(-1,7); ax.grid(alpha=0.3); ax.set_aspect('equal')

def draw_triangle(points, color, label):
    poly = Polygon(points, fill=False, color=color, linewidth=3, label=label)
    ax.add_patch(poly)
    for p in points:
        ax.plot(p[0],p[1],'o',color=color,markersize=8)

draw_triangle([A1,B1,C1], 'red',   '△ABC（原图）')
draw_triangle([A2,B2,C2], 'blue',  '△A′B′C′（平移+旋转后）')

ax.text(3, -0.8, '三边相等 → 全等！', fontsize=16, ha='center', color='purple', weight='bold')
ax.legend(); ax.set_title('全等三角形 SSS 判定现场演示')
plt.show()
```

#### 2. 相似三角形（概念 + 三大判定法 + 比例线）

**概念**  
形状一样，大小可以不一样（像照片放大缩小）

**三大判定口诀**  
- AAA（AA）：角角对应相等  
- SAS：两边成比例且夹角相等  
- SSS：三边成比例

```python
# AA 相似演示
fig, ax = plt.subplots(figsize=(12,7))
ax.set_xlim(0,12); ax.set_ylim(0,10); ax.grid(alpha=0.3); ax.set_aspect('equal')

# 小三角形
A,B,C = np.array([2,1]), np.array([8,1]), np.array([5,7])
# 大三角形（放大2倍）
A2,B2,C2 = A*2, B*2, C*2 - np.array([0,3])

draw_triangle([A,B,C],  'red',  '小△')
draw_triangle([A2,B2,C2],'blue','大△（放大2倍）')

ax.text(6,0.2,'∠A=∠A′, ∠B=∠B′ → AA相似，比例 1:2',fontsize=14,ha='center',color='purple')
ax.legend(); ax.set_title('相似三角形 AA 判定 + 比例关系')
plt.show()
```

#### 3. 圆的性质（初中几何之神）

2. 垂直于弦的直径平分弦  
3. 等弧所对的圆周角相等

#### 4. 几何变换四大金刚（矩阵版提前预习）

**概念讲解（初中生也能听懂）**  
- 平移：整个图形整体搬家  
- 旋转：绕一个点转圈圈  
- 轴对称：对着一条线翻过去  
- 中心对称：对着一个点转180°

```python
# 用矩阵实现四大变换（高中甚至大学才教，我们现在就玩！）
def translate(points, dx, dy):
    return points + np.array([dx, dy])

def rotate(points, angle_deg, center=(0,0)):
    theta = np.radians(angle_deg)
    c, s = np.cos(theta), np.sin(theta)
    R = np.array([[c,-s],[s,c]])
    return (R @ (points - center).T).T + center

def reflect_x(points):   # 关于 x 轴
    return points * np.array([1,-1])

def central_symmetry(points, center=(0,0)):
    return 2*center - points

# 原始三角形
P = np.array([[1,1], [5,1], [3,4]])

fig, axs = plt.subplots(2,2,figsize=(14,14))
axs = axs.ravel()

# 原图
axs[0].plot(*zip(*np.append(P,P[0:1],axis=0).T), 'ro-')
axs[0].set_title('原图'); axs[0].grid(); axs[0].set_aspect('equal')

# 平移
P1 = translate(P, 4, 2)
axs[1].plot(*zip(*np.append(P1,P1[0:1],axis=0).T), 'bo-')
axs[1].set_title('平移 (4,2)'); axs[1].grid(); axs[1].set_aspect('equal')

# 旋转90°
P2 = rotate(P, 90, center=(3,2))
axs[2].plot(*zip(*np.append(P2,P2[0:1],axis=0).T), 'go-')
axs[2].set_title('绕(3,2)旋转90°'); axs[2].grid(); axs[2].set_aspect('equal')

# 中心对称
P3 = central_symmetry(P, center=(3,2))
axs[3].plot(*zip(*np.append(P3,P3[0:1],axis=0).T), 'mo-')
axs[3].set_title('关于点(3,2)中心对称'); axs[3].grid(); axs[3].set_aspect('equal')

for ax in axs:
    ax.set_xlim(-2,10); ax.set_ylim(-2,10)
plt.suptitle('几何变换四大金刚（矩阵版）—— 初中提前玩大学内容',fontsize=20)
plt.tight_layout()
plt.show()
```
