#### 1. 等差数列（高中数列之王）

**高中必背公式（背会高考直接拿12分）**
- 第 n 项：aₙ = a₁ + (n−1)d  
- 前 n 项和：Sₙ = n/2 (a₁+aₙ) = n/2 [2a₁+(n−1)d]

```python
# 一键生成等差数列
a1, d = 3, 5
n = 20
arith = a1 + (np.arange(n) ) * d   # 注意 arange 从0开始
print(f"等差数列（首项 {a1}，公差 {d}）前 {n} 项：")
print(arith)

# 前 n 项和两种算法对比
Sn_formula = n/2 * (2*a1 + (n-1)*d)
Sn_sum = np.sum(arith)
print(f"S_{n} 公式法 = {Sn_formula}, 累加法 = {Sn_sum}")
```

#### 2. 等比数列（高中最美数列）

**高中必背公式**
- 第 n 项：aₙ = a₁·qⁿ⁻¹  
- 前 n 项和：Sₙ = a₁(1−qⁿ)/(1−q)   (q≠1)

```python
# 一键生成等比数列
a1, q = 2, 3
n = 15
geo = a1 * q**(np.arange(n))
print(f"等比数列（首项 {a1}，公比 {q}）前 {n} 项：")
print(geo.astype(int))

# 等比求和公式 vs 直接累加
Sn_formula = a1 * (1 - q**n) / (1 - q)
Sn_sum = np.sum(geo)
print(f"S_{n} 公式法 = {Sn_formula:.0f}, 累加法 = {Sn_sum:.0f}")
```

#### 5. 数学归纳法（高中压轴证明神器）

**高中最强模板（背会高考直接拿15分）**
1. n=1（或 n=n₀）时成立  
2. 假设 n=k 成立 → 推出 n=k+1 也成立  
∴ 对一切 n≥1 成立

```python
# 用 Python 现场验证 + 证明 1+2+…+n = n(n+1)/2
def sum_formula(n): return n*(n+1)//2

for n in range(1, 101):
    assert sum_formula(n) == np.sum(np.arange(1,n+1))
print("1+2+…+n = n(n+1)/2  对前100项全部验证通过")

# 归纳法证明 2ⁿ > n²（n≥5）
print("\n数学归纳法证明：2ⁿ > n² (n≥5)")
print("① n=5: 32 > 25 成立")
print("② 假设 n=k≥5 时 2ᵏ > k²")
print("   n=k+1: 2^{k+1} = 2·2ᵏ > 2k² ≥ k² + 2k + 1 = (k+1)² (因为 k≥5 → 2k² ≥ k²+2k+1)")
print("∴ 成立")
```
