#### 1. 直线与圆（基础却最容易丢分）

**终极公式（背会高考解析几何前两题直接拿满）**
- 直线：点斜式、两点式、一般式  
- 圆：标准式 (x−h)²+(y−k)²=r²，一般式 x²+y²+Dx+Ey+F=0

```python
# 现场求直线与圆交点个数（高考压轴预备）
print("直线 x+y−3=0 与圆 x²+y²=4 交点个数？")
# 联立方程 → 代入得 2x²−6x+5=0 → Δ=36−40<0 → 无交点
x = np.linspace(-3,3,400)
y1 = 3 - x
y2 = np.sqrt(4-x**2)
y3 = -np.sqrt(4-x**2)

plt.figure(figsize=(10,10))
plt.plot(x,y1,'red',linewidth=3,label='直线 x+y=3')
circle = Circle((0,0),2,color='blue',fill=False,linewidth=3)
plt.gca().add_patch(circle)
plt.axis('equal'); plt.grid(alpha=0.3); plt.legend()
plt.title('直线与圆：无交点（Δ<0）',fontsize=20)
plt.show()
```

#### 2. 圆锥曲线全家福（椭圆、双曲线、抛物线）

**统一第二定义 + 标准方程（必须倒背如流）**

| 曲线   | 标准方程                              | 离心率 e      | 第二定义（到定点与定直线距离比） |
|--------|---------------------------------------|---------------|----------------------------------|
| 椭圆   | x²/a² + y²/b² = 1 (a>b)               | e=c/a <1      | 距离之比 = e <1                  |
| 双曲线 | x²/a² − y²/b² = 1                     | e=c/a >1      | 距离之比 = e >1                  |
| 抛物线 | y² = 4px                              | e=1           | 距离之比 = 1                     |

```python
# 一张图看懂所有圆锥曲线
fig, axs = plt.subplots(2,2,figsize=(16,14))

x = np.linspace(-4,4,400)

# 椭圆
axs[0,0].plot(x, 3*np.sqrt(1-x**2/16), 'red')
axs[0,0].plot(x,-3*np.sqrt(1-x**2/16), 'red')
axs[0,0].set_title('椭圆 x²/16 + y²/9 =1  (e=√7/4≈0.66)',fontsize=16)

# 双曲线
axs[0,1].plot(x, 3*np.sqrt(x**2/4-1), 'blue')
axs[0,1].plot(x,-3*np.sqrt(x**2/4-1), 'blue')
axs[0,1].set_title('双曲线 x²/4 − y²/9 =1  (e=√13/2≈1.8)',fontsize=16)

# 抛物线
axs[1,0].plot(x, x**2/4, 'green',linewidth=3)
axs[1,0].set_title('抛物线 y²=4x  (e=1)',fontsize=16)

# 退化情况
axs[1,1].axis('off')
axs[1,1].text(0.5,0.8,'当 e=0 → 圆\n当 e=1 → 抛物线\n当 e>1 → 双曲线\n当 e→∞ → 两条直线', 
             transform=axs[1,1].transAxes, fontsize=20, ha='center')

plt.suptitle('圆锥曲线全家福 + 离心率 e 统一（高中解析几何之魂）',fontsize=24)
plt.tight_layout()
plt.show()
```

#### 3. 极坐标与参数方程（高考新热点+奥赛神器）

**经典极坐标方程（必须会背会画）**
- 玫瑰线：r = a cos(nθ) 或 r = a sin(nθ)  
- 阿基米德螺线：r = aθ  
- 心形线：r = a(1−cosθ)

```python
# 画四种极坐标经典曲线
theta = np.linspace(0, 2*np.pi, 1000)

plt.figure(figsize=(16,12))

plt.subplot(221, projection='polar')
r1 = 3*np.cos(4*theta)
plt.polar(theta, np.abs(r1), 'red', linewidth=2)
plt.title('玫瑰线 r=3cos(4θ)（8瓣）')

plt.subplot(222, projection='polar')
r2 = theta
plt.polar(theta, r2, 'blue', linewidth=2)
plt.title('阿基米德螺线 r=θ')

plt.subplot(223, projection='polar')
r3 = 2*(1-np.cos(theta))
plt.polar(theta, r3, 'purple', linewidth=3)
plt.title('心形线 r=2(1−cosθ)')

plt.subplot(224, projection='polar')
r4 = 1/(1-0.5*np.cos(theta))
plt.polar(theta, r4, 'green', linewidth=3)
plt.title('圆锥曲线极坐标统一：r=ed/(1−e cosθ)\ne=0.5→椭圆')

plt.suptitle('极坐标四大名曲线（高考+奥赛必考）',fontsize=24)
plt.tight_layout()
plt.show()
```

#### 4. 参数方程（动点轨迹终极武器）

```python
# 经典参数方程：摆线（车轮滚动轨迹）
t = np.linspace(0, 4*np.pi, 1000)
a = 1
x = a*(t - np.sin(t))
y = a*(1 - np.cos(t))

plt.figure(figsize=(12,6))
plt.plot(x,y,'purple',linewidth=3)
plt.plot(x[::50],y[::50],'o',color='red',markersize=8)
plt.title('摆线（车轮滚动轨迹）参数方程：x=a(t−sin t), y=a(1−cos t)',fontsize=18)
plt.axis('equal'); plt.grid(alpha=0.3)
plt.show()
```
