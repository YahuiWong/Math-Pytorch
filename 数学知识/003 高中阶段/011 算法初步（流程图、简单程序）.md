#### 2. 三大经典算法（高考必考 + 信息学入门）

**算法1：辗转相除法求最大公约数（欧几里得算法）**
```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a

print(gcd(48, 18))   # 6
```

**算法2：秦九韶算法（一分钟多项式求值）**
```python
# 求 p(x)=3x⁴+5x³−2x²+7x−10 在 x=2 的值
def horner(coeffs, x):
    result = coeffs[0]
    for c in coeffs[1:]:
        result = result * x + c
    return result

print(horner([3,5,-2,7,-10], 2))   # 108
```

**算法3：埃氏筛法求100以内素数**
```python
def sieve(n):
    is_prime = [True]*(n+1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(n**0.5)+1):
        if is_prime[i]:
            for j in range(i*i, n+1, i):
                is_prime[j] = False
    return [i for i in range(2,n+1) if is_prime[i]]

print("100以内素数个数 =", len(sieve(100)))   # 25个
```
