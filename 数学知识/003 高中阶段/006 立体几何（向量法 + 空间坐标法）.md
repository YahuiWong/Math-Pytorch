#### 1. 空间向量基本功（高中立体几何降维打击武器）

**高中最强三板斧（背会高考立体几何直接躺赢）**
1. 向量叉积 → 垂直于两向量的法向量  
2. 数量积 → 夹角、二面角  
3. 混合积 → 体积

```python
# 现场演示三板斧
a = np.array([1,2,3])
b = np.array([4,5,6])
c = np.array([1,1,-1])

print("向量三板斧现场表演".center(60,"="))
print(f"叉积 a×b = {np.cross(a,b)}")                     # 法向量
print(f"数量积 a·c = {np.dot(a,c)}")                     # 夹角余弦
print(f"混合积 [a,b,c] = {np.dot(a,np.cross(b,c)):.0f}") # 体积
```

#### 2. 空间直线与平面（向量法终极秒杀）

**最强记忆口诀**
- 直线：点 + 方向向量  
- 平面：点 + 法向量 或 点 + 两个不共线向量

```python
# 例题：求异面直线距离（高考最难题型）
print("高考最难：两条异面直线距离（向量法一招秒）")

# 直线 L1：r = <1,0,0> + t<1,1,1>
# 直线 L2：r = <0,1,1> + s<1,0,1>

P1 = np.array([1,0,0]); d1 = np.array([1,1,1])
P2 = np.array([0,1,1]); d2 = np.array([1,0,1])

n = np.cross(d1,d2)                    # 公共法向量
distance = np.abs(np.dot(P2-P1, n)) / np.linalg.norm(n)
print(f"两异面直线距离 = {distance:.3f}")   # 1/√3

# 画图让你彻底信服
fig = plt.figure(figsize=(10,10))
ax = fig.add_subplot(111, projection='3d')
t = s = np.linspace(-2,2,50)
ax.plot(P1[0]+t*d1[0], P1[1]+t*d1[1], P1[2]+t*d1[2], 'red', linewidth=4, label='L1')
ax.plot(P2[0]+s*d2[0], P2[1]+s*d2[1], P2[2]+s*d2[2], 'blue', linewidth=4, label='L2')
ax.set_xlabel('x'); ax.set_ylabel('y'); ax.set_zlabel('z')
ax.legend(); ax.set_title('异面直线距离（向量法一秒求出）')
plt.show()
```

#### 3. 二面角（向量法最优雅解法）

**终极公式（背会高考二面角直接拿满分）**
二面角 = 两个法向量的夹角（或补角，取锐角或题设）

```python
# 例：求平面 x+y+z=1 与 x-y+z=1 的二面角
n1 = np.array([1,1,1])
n2 = np.array([1,-1,1])
cos_phi = np.abs(np.dot(n1,n2)) / (np.linalg.norm(n1)*np.linalg.norm(n2))
phi = np.rad2deg(np.arccos(cos_phi))
print(f"二面角 = {phi:.1f}°")   # 54.7°
```

#### 4. 空间坐标法经典题型全家福

```python
# 球面距离（高考新热点）
print("球面距离（2024高考新题型）")
A = np.array([1,1,1]); B = np.array([1,-1,-1])
R = np.linalg.norm(A)   # 半径 √3
cos_theta = np.dot(A,B) / (R*R)
great_circle = R * np.arccos(cos_theta)
print(f"球面上 A 到 B 的最短距离 = {great_circle:.3f}")
```

#### 5. 空间几何终极总结表（高考+奥赛满分表）

| 题型             | 向量法公式（秒杀）                                 | 坐标法公式                     | 难度等级 |
|------------------|----------------------------------------------------|--------------------------------|----------|
| 异面直线距离     | |P₂-P₁|·|d₁×d₂| / |d₁×d₂|                           | 传统三垂线繁琐                 | ★★★★★    |
| 直线与平面夹角   | sinθ = |d·n| / (|d||n|)                               | 投影法                        | ★★★★     |
| 二面角           | cosφ = |n₁·n₂| / (|n₁||n₂|)（取锐角或补角）           | 法向量夹角                     | ★★★★★    |
| 点到平面距离     | |Ax₀+By₀+Cz₀+D| / √(A²+B²+C²)                      | 公式直接代                     | ★★       |
| 空间四边形对角线 | 向量加减                                           | 坐标直接算                     | ★★★      |
| 球面距离         | R·arccos(a·b / R²)                                 | 向量夹角                       | ★★★★★    |
