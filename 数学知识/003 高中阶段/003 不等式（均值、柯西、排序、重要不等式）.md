#### 1. 均值不等式（高考不等式之魂，背会直接拿20分）

**高中终极口诀（必须倒背如流）**
1. 算术 ≥ 几何 ≥ 调和 ≥ 最小  
2. 只有全相等时取等号  
3. AM-GM：(a+b)/2 ≥ √(ab)  
4. 推广：(a₁+a₂+…+an)/n ≥ n√(a₁a₂…an)

```python
# 现场验证 AM-GM
a = np.array([1,2,3,4,12])
AM = np.mean(a)
GM = np.prod(a)**(1/len(a))
print(f"5个数 {a}")
print(f"算术平均 = {AM:.4f}")
print(f"几何平均 = {GM:.4f}")
print(f"验证 AM ≥ GM：{AM >= GM}，相差 {AM-GM:.4f}")

# 经典高考题：a,b>0，a+b=1，求 ab 最大值
print("\n高考经典：a+b=1，a,b>0，求 ab 最大值")
print("由 AM-GM：(a+b)/2 ≥ √(ab) → 1/2 ≥ √(ab) → ab ≤ 1/4")
print("当且仅当 a=b=1/2 时取等 → 最大值 1/4")
```

#### 2. 柯西-施瓦茨不等式（Cauchy-Schwarz）—— 高中压轴最强武器

**高中最常用两种形式（必须会背会用）**
1. (a₁²+b₁²)(a₂²+b₂²) ≥ (a₁b₁ + a₂b₂)²  
2. (∑aᵢ²)(∑bᵢ²) ≥ (∑aᵢbᵢ)²  
3. 向量形式：|u·v| ≤ ||u||·||v||

```python
# 现场验证柯西
a = np.array([1,2,3])
b = np.array([4,5,6])
left  = np.sum(a**2) * np.sum(b**2)
right = (np.dot(a,b))**2
print(f"柯西验证：{left} ≥ {right} → {left >= right}")

# 高考最爱变形：求 (1+4+9)(1/1 + 1/4 + 1/9) 最小值
print("\n高考压轴：求 (1+4+9)(1+1/4+1/9) 最小值")
x = np.array([1,2,3])
y = np.array([1,1/2,1/3])
print(f"直接柯西：≥ (1*1 + 2*1/2 + 3*1/3)² = (1+1+1)² = 9")
print("当且仅当 x=y/k 时取等 → 最小值 9")
```

#### 3. 排序不等式（重排不等式）—— 高中不等式第二神器

**核心结论（背会秒杀选择填空）**
- 同序和最大，反序和最小  
- a₁≤a₂≤…≤an，b₁≤b₂≤…≤bn  
- 则 a₁bₙ + a₂bₙ₋₁ + … ≤ a₁b₁ + a₂b₂ + … ≤ a₁bₙ + a₂bₙ₋₁ + …

```python
a = np.array([1,2,3,4])
b = np.array([10,20,30,40])

same_order = np.sum(a*b)
reverse_order = np.sum(a[::-1]*b)
random_order = np.sum(np.random.permutation(a)*b)

print(f"同序和（最大）= {same_order}")
print(f"乱序和       = {random_order}")
print(f"反序和（最小）= {reverse_order}")
```

#### 4. 重要不等式全家福 + 放缩法

| 不等式           | 表达式                                     | 取等条件             | 高考经典用途                     |
|------------------|--------------------------------------------|----------------------|----------------------------------|
| 伯努利             | (1+x)ⁿ ≥ 1+nx (x>-1,n∈ℕ⁺)                 | x=0 或 n=1           | 放缩求极限、证明不等式           |
| 琴生不等式         | a³+b³+c³-3abc ≥ (a+b+c)(a-b)²+(b-c)²+(c-a)²)/2 | a=b=c                | 三元最值                         |
| 绝对值三角不等式   | |a+b| ≤ |a|+|b|                            | 任意实数             | 估值、放缩                       |
| 切比雪夫           | 同增序列同序和最大                         | —                    | 排序不等式加强版                 |

```python
# 伯努利不等式现场验证
x = 0.5
n = np.arange(1,11)
left  = (1+x)**n
right = 1 + n*x
plt.plot(n, left, 'o-', label='(1+0.5)^n')
plt.plot(n, right, 's--', label='1 + n×0.5')
plt.legend(); plt.grid(); plt.title('伯努利不等式：(1+x)^n ≥ 1+nx')
plt.show()
```

#### 5. 高考压轴终极放缩法演示（2024年全国卷难度）

**题目**：已知 a,b,c>0，求证：a/(b+c) + b/(a+c) + c/(a+b) ≥ 3/2

```python
print("高考压轴不等式证明现场直播".center(60,"="))
print("方法一：柯西直接秒")
print("由柯西：∑ a/(b+c) ≥ (1+1+1)² / ∑(b+c)/a = 9 / (∑b/a + ∑c/a)")
print("        = 9 / ( (b+a+c)/a + (a+b+c)/b + (a+b+c)/c ) wait… 换个思路")

print("\n方法二：调和均值法（最优）")
print("a/(b+c) = a/( (a+b+c) - a ) ≥ a / (2(a+b+c)/3 * 3 - a) 太复杂")

print("\n方法三：经典换元 + 均值（标准答案）")
print("令 s=a+b+c，则")
print("a/(b+c) = a/(s-a) = (s - (s-a))/ (s-a) = s/(s-a) - 1")
print("∑ a/(b+c) = ∑ (s/(s-a) - 1) = s (1/(s-a)+1/(s-b)+1/(s-c)) - 3")
print("由 HM ≤ AM：3 / (1/(s-a)+1/(s-b)+1/(s-c)) ≤ ( (s-a)+(s-b)+(s-c) )/3 = (3s - s)/3 = 2s/3")
print("→ 1/(s-a)+1/(s-b)+1/(s-c) ≥ 9/(2s)")
print("→ 原式 ≥ s·(9/(2s)) - 3 = 9/2 - 3 = 3/2")
print("得证！当且仅当 a=b=c 时取等")
```
