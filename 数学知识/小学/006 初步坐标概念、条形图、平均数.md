#### 1. 初步坐标概念（小学五年级第一次见到坐标系 → 现代数学的“原点”）

**小学生概念**  
一张纸上画两条互相垂直的线，一条横着一边竖着，交叉的地方叫原点 O，所有位置都可以用“向右几、向上几”来表示。

```python
# 用 PyTorch 张量表示小学坐标纸上的点
A = torch.tensor([2.0, 1.0])   # 向右2，向上1
B = torch.tensor([5.0, 4.0])
C = torch.tensor([1.0, 5.0])
D = torch.tensor([6.0, 2.0])

points = torch.stack([A, B, C, D])
labels = ['A', 'B', 'C', 'D']

# 现场画出小学坐标纸
plt.figure(figsize=(10,9))
plt.grid(True, alpha=0.4)
plt.axhline(0, color='black', lw=1)
plt.axvline(0, color='black', lw=1)
plt.scatter(points[:,0], points[:,1], c='red', s=100, zorder=5)

for i, label in enumerate(labels):
    plt.text(points[i,0]+0.2, points[i,1], label, fontsize=16, weight='bold')

plt.xlim(0, 7); plt.ylim(0, 6)
plt.xticks(range(8)); plt.yticks(range(7))
plt.title('小学第一次见到的坐标系（其实就是 ℝ² 张量空间）', fontsize=18)
plt.xlabel('向右 →'); plt.ylabel('向上 →')
plt.show()
```

#### 2. 条形图（小学统计第一课 → 现代数据可视化原生形态）

**小学生概念**  
把不同东西的数量用长短不一的条条表示，一眼看出谁多谁少

```python
# 六年级经典题目：五（1）班同学最喜欢的运动统计
sports = ['足球', '篮球', '羽毛球', '乒乓球', '游泳']
votes  = torch.tensor([18, 25, 15, 30, 12])

               edgecolor='black', linewidth=1.2)

for bar in bars:
    height = bar.get_height()
    plt.text(bar.get_x() + bar.get_width()/2., height + 0.5,
             f'{int(height)}人', ha='center', va='bottom', fontsize=14, weight='bold')

plt.ylabel('人数', fontsize=14)

```

#### 3. 平均数（小学统计之魂 → 深度学习损失函数的祖宗）

**小学生概念**  
把总和平均分，每个人得到一样多，就是平均数

```python
# 经典应用题：5个同学身高分别是 142cm、148cm、135cm、150cm、145cm，平均身高多少？
heights = torch.tensor([142.0, 148.0, 135.0, 150.0, 145.0])

# 方法①：小学教的算法
total = heights.sum()
avg_hand = total / 5

# 方法②：PyTorch 一键（和深度学习训练一模一样）
avg_torch = heights.mean()

print(f"总和 = {total.item()} cm")
print(f"平均身高（手算）= {avg_hand.item():.1f} cm")
print(f"平均身高（PyTorch）= {avg_torch.item():.1f} cm")

# 深度学习视角：平均数就是最小化 L2 损失！
# 设平均身高为 x，最小化 Σ(xi − x)² → 解就是 x = mean
x = torch.tensor(144.0, requires_grad=True)  # 随便猜一个
loss = ((heights - x) ** 2).mean()           # MSE 损失！
loss.backward()
print(f"如果猜 144cm，梯度 = {x.grad.item():.2f} ← 告诉你要往上调！")
```

#### 4. 小学终极压轴：坐标 + 统计 + 平均数 混合大招

```python
# 题目：某校五个年级平均身高统计（坐标 + 条形图 + 平均数）
grades = ['一年级', '二年级', '三年级', '四年级', '五年级']
avg_heights = torch.tensor([118.5, 125.3, 132.7, 139.2, 146.8])

# 双图神级结合（小学作业进化版）
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(16,7))

# 左图：条形图
bars = ax1.bar(grades, avg_heights, color='skyblue', edgecolor='navy', alpha=0.8)
for bar in bars:
    h = bar.get_height()
    ax1.text(bar.get_x()+bar.get_width()/2, h+0.5, f'{h}cm', ha='center', fontsize=12)
ax1.set_title('各年级平均身高条形图', fontsize=16)
ax1.set_ylabel('平均身高 (cm)')

# 右图：折线图 + 散点（坐标系）
years = torch.arange(1, 6)
ax2.plot(years, avg_heights, 'o-', color='red', linewidth=3, markersize=10)
for i, h in enumerate(avg_heights):
    ax2.text(years[i], h+1.5, f'{h}cm', ha='center', fontsize=12, weight='bold')
ax2.set_xticks(years)
ax2.set_xticklabels(grades)
ax2.grid(True, alpha=0.3)
ax2.set_title('各年级平均身高趋势（坐标折线图）', fontsize=16)

plt suptitle('小学统计终极形态：条形图 + 坐标图 + 平均数', fontsize=22, y=1.02)
plt.tight_layout()
plt.show()

# 全校平均身高（平均数的平均数）
school_avg = avg_heights.mean()
print(f"全校平均身高 = {school_avg:.2f} cm")
```
