### 小学阶段第四块：简单几何（线段、角、三角形、矩形、圆、周长面积体积）  
全部用 PyTorch + 严格数学定义 + 可运行代码 + 小学生能听懂的概念描述

```python
import torch
import numpy as np
import matplotlib.pyplot as plt
from math import pi, sin, cos, radians
torch.manual_seed(2025)
```

#### 1. 线段与点（几何的原子）

**小学生概念**  
点没有大小，线段是两点之间最短的直线距离

```python
# 用 PyTorch 张量表示平面上的点（小学四年级就该这样教！）
A = torch.tensor([0.0, 0.0])
B = torch.tensor([6.0, 0.0])
C = torch.tensor([0.0, 8.0])
D = torch.tensor([6.0, 8.0])

# 线段长度（小学公式：|b-a|）
def dist(p1, p2):
    return torch.norm(p2 - p1)          # 自动开根号

print(f"线段 AB 长度 = {dist(A,B):.1f}")     # 6.0
print(f"线段 AC 长度 = {dist(A,C):.1f}")     # 8.0
print(f"线段 AD 长度 = {dist(A,D):.1f}")     # 10.0（勾股定理！）
```

#### 2. 角（小学最容易画歪的）

**概念**  
角是由一条射线绕端点转到另一条射线形成的

```python
# 用角度制表示角（小学生最熟悉）
angle_30  = torch.tensor(30.0)   # 30°
angle_90  = torch.tensor(90.0)   # 直角
angle_180 = torch.tensor(180.0)  # 平角

# 弧度制（大学才教，但现在偷偷告诉你）
print("30° 等于", radians(30), "弧度")
print("90° 等于", radians(90), "弧度（π/2）")
```

#### 3. 三角形（小学几何之王）

**核心公式全家福**

```python
# 三角形三边
a, b, c = 3.0, 4.0, 5.0

# 周长
perimeter = a + b + c
print(f"三边 {a},{b},{c} 的三角形周长 = {perimeter}")

# 面积：海伦公式（小学奥数最爱）
p = perimeter / 2
area_heron = torch.sqrt(p*(p-a)*(p-b)*(p-c))
print(f"海伦公式面积 = {area_heron:.2f}")   # 6.00

# 面积：底×高÷2（更简单）
area_simple = 0.5 * a * b
print(f"直角三角形面积 = {area_simple:.2f}")   # 6.00

# 验证：是不是直角三角形？
print("勾股定理验证：", torch.isclose(a**2 + b**2, c**2))   # True
```

#### 4. 矩形与正方形

```python
length = 7.0
width  = 5.0

# 周长与面积
rect_perim = 2 * (length + width)
rect_area  = length * width
print(f"长 {length}、宽 {width} 的矩形：")
print(f"   周长 = {rect_perim}，面积 = {rect_area}")

# 正方形（特例）
side = 6.0
print(f"边长 {side} 的正方形：周长 {4*side}，面积 {side**2}")
```

#### 5. 圆（小学最难记的公式）

```python
r = torch.tensor(5.0)          # 半径

# 周长（圆周长）
circumference = 2 * pi * r
print(f"半径 {r.item()} 的圆周长 ≈ {circumference:.2f}")

# 面积
area = pi * r**2
print(f"圆面积 ≈ {area:.2f}")

# 圆周率 π 的多种写法（小学生提前开眼界）
print("π 的几种来源：")
print("  torch.pi      =", torch.pi)
print("  numpy.pi      =", np.pi)
print("  22/7          =", 22/7)
print("  355/113       =", 355/113)      # 古代中国人祖冲之的精度！
```

#### 6. 体积（小学五年级最酷的部分）

```python
# 长方体
l, w, h = 4.0, 5.0, 6.0
volume_box = l * w * h
print(f"长方体 {l}×{w}×{h} 的体积 = {volume_box}")

# 正方体
s = 3.0
print(f"正方体边长 {s}，体积 = {s**3}")

# 圆柱（小学六年级压轴）
r_cyl = 3.0
h_cyl = 10.0
volume_cylinder = pi * r_cyl**2 * h_cyl
print(f"底面半径 {r_cyl}、高 {h_cyl} 的圆柱体积 ≈ {volume_cylinder:.2f}")
```

#### 7. 小学生画图神器（用 PyTorch + matplotlib 现场画）

```python
# 画一个带坐标轴的三角形、矩形、圆
fig, ax = plt.subplots(figsize=(8,8))
ax.set_xlim(-1, 8)
ax.set_ylim(-1, 9)
ax.grid(True, alpha=0.3)
ax.set_aspect('equal')

# 三角形 ABC
triangle = plt.Polygon([[0,0], [6,0], [0,8]], fill=None, edgecolor='red', linewidth=2, label='三角形')
ax.add_patch(triangle)

# 矩形
rect = plt.Rectangle((6,0), 2, 5, fill=None, edgecolor='blue', linewidth=2, label='矩形')
ax.add_patch(rect)

# 圆（圆心在 (4,6)，半径 2）
circle = plt.Circle((4, 6), 2, fill=None, edgecolor='green', linewidth=2, label='圆')
ax.add_patch(circle)

ax.legend()
ax.set_title("小学几何全家福（PyTorch + matplotlib 现场绘制）", fontsize=14)
plt.show()
```
